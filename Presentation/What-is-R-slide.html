What is R ?

  • R is the language and environment for statistical computing and graphics.
  • R provides a wide variety of statistical and graphical techniques, and is highly extensible.
  • data manipulation, calculation and graphical display
  • Founded by Ross Ihaka and Robert Gentleman in 1991.
  • It contains function and open source packages that facilitate the user to computation goals

What R can do ?

Some examples are

  • an effective data handling and storage facility,
  • a suite of operators for calculations on arrays, in particular matrices,
  • a large, coherent, integrated collection of intermediate tools for data analysis
  • graphical facilities for data analysis and display either on-screen or on hard copy,
  • a well-developed, simple and effective programming language which includes conditionals, loops, user-defined recursive functions and input and output facilities.
